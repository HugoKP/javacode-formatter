javacode-styler

javacode-styler.global
Contém a classe Global com campos e métodos para acesso por todas as classes do projeto.
javacode-styler.tokens
Contém a classe abstrata Tokens e suas subclasses encarregadas de analisar cada tipo de token a ser estilizado em um código fonte Java.
